Certifications Développement Web

Voici les certifications techniques que j’ai obtenues dans le cadre de mon parcours en développement web. Elles démontrent mes compétences pratiques en front-end, back-end et gestion de bases de données, des compétences recherchées par les employeurs pour une alternance en développement web.

JavaScript Basic

Maîtrise des bases du JavaScript : variables, fonctions, logique, manipulation du DOM.

NodeJS Basic

Développement back-end avec JavaScript : serveur, API et logique côté serveur.

SQL Basic

Gestion des bases de données relationnelles : requêtes SQL, CRUD et structuration des données.

CSS Basic

Création d’interfaces modernes, responsive design et mise en forme avancée.

Angular Basic

Utilisation d’un framework front-end pour créer des applications web dynamiques.

Python Basic

Maîtrise des bases de Python : variables, structures de données, fonctions et logique de programmation.
